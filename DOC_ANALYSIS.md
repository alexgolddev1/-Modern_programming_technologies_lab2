# Анализ `СТП ЧНУ лабораторна 2.doc`

Документ удалось прочитать через разбор OLE-структуры `.doc` и извлечение текста из потока `WordDocument`, а кодовых фрагментов из встроенных `EmbeddedOdf`-объектов.

Ключевые требования из лабораторной:

1. Создать Django-проект и приложение.
2. Подключить SQLite и выполнить миграции.
3. Настроить административную панель.
4. Реализовать CRUD для сущности заказов.
5. Подготовить HTML-шаблоны для создания, просмотра, редактирования и удаления записей.

Кодовые фрагменты из документа описывают проект `crudproject` и приложение `crudapp` со следующей моделью:

- `Orders`
- поля: `oid`, `fname`, `lname`, `price`, `mail`, `addr`

В документе также приведены:

- `ModelForm` для `Orders`
- функции-представления для create/read/update/delete
- шаблоны `layout.html`, `order.html`, `show.html`, `confirmation.html`
- маршруты приложения и корневого проекта

Собранный в этом каталоге проект следует этой структуре, но использует актуальные Django-настройки и немного более чистые URL.
